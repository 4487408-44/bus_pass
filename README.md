# Bus Pass Management System

## Description
This is a web-based bus pass system developed using Python and SQLite.

## Technologies Used
- Python
- SQLite
- HTML, CSS, JavaScript

## How to Run
1. Open terminal
2. Go to project folder
3. Run:
   python server.py
4. Open browser:
   http://localhost:8082

## Features
- User Registration
- Login System
- Bus Pass Application
- Data stored in SQLite database
